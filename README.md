# 📚 Estudos em Spring Boot

Repositório dedicado aos meus estudos e experimentos com o framework **Spring Boot**, abordando desde conceitos fundamentais até implementações mais avançadas.

## 🚀 Tecnologias Utilizadas

- Java 17+
- Spring Boot 3.x
- Spring Data JPA
- Spring Web (REST)
- Maven / Gradle
- Banco de dados (H2, MySQL, PostgreSQL)

## 📖 Conteúdos Abordados

- [ ] Configuração e estrutura de um projeto Spring Boot
- [ ] Criação de endpoints REST (GET, POST, PUT, DELETE)
- [ ] Injeção de dependência e inversão de controle (IoC)
- [ ] Persistência com Spring Data JPA e Hibernate
- [ ] Spring Security e JWT
- [ ] Testes com JUnit e Mockito
- [ ] Deploy com Docker

## ▶️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/spring-boot-studies.git

# Acesse a pasta do projeto desejado
cd spring-boot-studies/02-rest-api

# Execute com Maven
./mvnw spring-boot:run
```

Este repositório serve como portfólio de aprendizado pessoal, documentando a evolução nos estudos do ecossistema Spring.
